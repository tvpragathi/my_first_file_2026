## Issues Faced & Solutions

### 1. Extension login failed (Claude Code & Codex)
- **Cause:** Node.js was not detected due to PATH misconfiguration  
- **Solution:** Installed Node.js, updated PATH environment variable, restarted Cursor IDE, and re-logged in

---

### 2. Git push error (src refspec main does not match any)
- **Cause:** No initial commit was created  
- **Solution:** Added files using `git add .`, created initial commit, and pushed again

---

### 3. Pathspec error while adding files
- **Cause:** Files were not present or not detected in the directory  
- **Solution:** Created/added files in correct folder and verified using `git status`
